# Politique de Confidentialité - TradingCourseArena

**Dernière mise à jour :** 15 janvier 2026

## Introduction

TradingCourseArena ("nous", "notre", "l'application") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles.

## Collecte de Données

### Données Collectées

L'application TradingCourseArena collecte les données suivantes :

1. **Informations de Compte**
   - Adresse email
   - Mot de passe (crypté)
   - Nom d'utilisateur (si fourni)

2. **Données de Progression**
   - Scores et statistiques
   - Historique des questions répondues
   - Temps passé sur chaque question
   - Progression par catégorie (calcul mental, probabilités, etc.)

3. **Données d'Utilisation**
   - Sessions d'entraînement
   - Préférences utilisateur
   - Données de performance

### Données NON Collectées

Nous ne collectons **PAS** :
- Informations de localisation précises
- Contacts de votre appareil
- Photos ou fichiers multimédias
- Données de paiement (l'app est gratuite)

## Utilisation des Données

Vos données sont utilisées uniquement pour :

1. **Fonctionnalités de l'Application**
   - Sauvegarder votre progression
   - Générer des statistiques personnalisées
   - Améliorer votre expérience utilisateur

2. **Amélioration du Service**
   - Analyser les performances globales
   - Identifier les bugs et problèmes
   - Développer de nouvelles fonctionnalités

3. **Communication**
   - Envoyer des notifications importantes (si activées)
   - Répondre à vos demandes de support

## Stockage des Données

Toutes les données sont stockées de manière sécurisée sur :
- **Supabase** : Infrastructure cloud sécurisée avec chiffrement SSL/TLS
- **Base de données** : PostgreSQL avec authentification sécurisée
- **Authentification** : Gestion sécurisée des mots de passe (hachage bcrypt)

## Partage des Données

**Nous ne partageons PAS vos données personnelles avec des tiers**, sauf :
- Si requis par la loi
- Pour protéger nos droits légaux
- En cas de fusion ou acquisition (avec notification préalable)

## Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :
- Chiffrement des données en transit (HTTPS)
- Authentification sécurisée
- Accès restreint aux données
- Sauvegardes régulières

## Vos Droits

Vous avez le droit de :

1. **Accéder à vos données**
   - Consulter toutes vos données dans l'application
   - Exporter vos statistiques

2. **Modifier vos données**
   - Mettre à jour votre profil
   - Modifier vos préférences

3. **Supprimer vos données**
   - Supprimer votre compte à tout moment
   - Toutes vos données seront supprimées de manière permanente

4. **Désactiver votre compte**
   - Suspendre temporairement votre compte
   - Réactiver à tout moment

## Cookies et Technologies Similaires

L'application utilise des technologies de stockage local (AsyncStorage) pour :
- Sauvegarder vos préférences
- Maintenir votre session
- Améliorer les performances
